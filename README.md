# Trading Indicator — TradingView Pine Script

A TradingView Pine Script indicator designed for **Nasdaq (NQ)** and **S&P 500 (ES)** futures trading analysis.

## Overview

This indicator provides technical analysis tools tailored for intraday and swing trading of Nasdaq and SPX futures contracts. It is built using Pine Script and runs directly inside the TradingView platform.

## Features

- Custom signal generation for NQ and ES futures
- Visual overlays for key support and resistance levels
- Trend direction and momentum analysis
- Configurable alerts for entry and exit conditions
- Optimized for futures market sessions (CME trading hours)

## Instruments

| Instrument | Exchange | Description |
|---|---|---|
| NQ1! | CME | Nasdaq 100 E-mini Futures |
| ES1! | CME | S&P 500 E-mini Futures |
| MNQ1! | CME | Micro Nasdaq 100 Futures |
| MES1! | CME | Micro S&P 500 Futures |

## Requirements

- [TradingView](https://www.tradingview.com) account (Free or Pro)
- Pine Script v5

## Usage

1. Open TradingView and navigate to the Pine Script Editor (`/` → Pine Editor).
2. Copy the contents of the `.pine` script file.
3. Paste into the editor and click **Add to chart**.
4. Configure the indicator settings to match your trading style and timeframe.

## Recommended Timeframes

- **1m / 3m / 5m** — Scalping
- **15m / 30m** — Intraday swing
- **1H / 4H** — Positional analysis
